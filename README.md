# 🏚️ Дом Палача — 2D пиксельный хоррор

Сбеги из дома маньяка Сайласа за 5 ночей. Прячься, ищи предметы, крафти оружие, выживай.

## 🎮 Возможности

- **Одиночная игра**: 5 уровней сложности, 4 ночи побега
- **Онлайн-мультиплеер** через Supabase (до 4 игроков)
- **LAN-мультиплеер** через `server.js` (для игры в одной Wi-Fi сети)
- **Туториал** для новичков (интерактивный)
- **Крафтинг**: молотов, кислотная бомба, электро-капкан, бомба
- **Достижения** (20 ачивок) и статистика
- **Адаптивная музыка** меняется по ситуации
- **Кот Сайласа** — наступишь, он мяукнет
- **Сны маньяка**, зеркала, кровавые следы
- Адаптивный UI (PC / Mobile / Gamepad)

## 🚀 Как запустить

### Просто играть (без онлайна)
Открой `index.html` в браузере.

### С LAN-сервером (для друзей в одной Wi-Fi)
```bash
npm install
npm start
```
Открой `http://localhost:8080`. Друзья в той же Wi-Fi видят сервер.

### С онлайном через интернет
1. Настрой Supabase — следуй `SUPABASE_SETUP.md`
2. Запусти игру — онлайн работает автоматически

## 📱 Сборка APK для Android

См. файл `android-build/` — там настроено через **Capacitor**.

GitHub Actions автоматически собирает APK при каждом push (см. `.github/workflows/build-apk.yml`).

Готовый APK можно скачать из **Actions → последний run → Artifacts**.

## 🗂 Структура

| Файл | Описание |
|------|----------|
| `index.html` | Игра целиком (HTML+CSS+JS, ~490 КБ) |
| `server.js` | LAN-сервер (Node.js + WebSocket + mDNS) |
| `manifest.json` | PWA-манифест |
| `service-worker.js` | Офлайн-кэш PWA |
| `icon-*.png` | Иконки приложения |
| `screenshot-*.png` | Скриншоты для магазина |
| `android-build/` | Capacitor-конфиг для APK |
| `.github/workflows/build-apk.yml` | Автоматическая сборка APK |
| `SUPABASE_SETUP.md` | Инструкция по настройке БД |

## 🎯 Управление

**ПК**: WASD движение, Shift бег, Ctrl красться, E взаимодействие, F использовать, Q выбросить, C крафтинг, T фонарик, Esc пауза

**Маньяк в онлайне**: E удар, R капкан, V видение жертв сквозь стены

**Мобильный**: виртуальный джойстик слева + кнопки справа
