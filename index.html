СОЗДАНИЕ КОМНАТЫ

Для создания комнаты нужно запустить серверную часть на своём компьютере.
Откройте папку с игрой в терминале и выполните: 

npm install
npm start

После этого в этом меню появится ваша комната. Друзья в той же Wi-Fi сети увидят её автоматически.

Подсказка: настроить имя комнаты можно так:
ROOM_NAME="Моя крутая комната" npm start
